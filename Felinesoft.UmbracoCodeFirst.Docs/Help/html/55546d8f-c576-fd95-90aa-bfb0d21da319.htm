CodeFirstDocumentViewTdocument, Tviewmodel Class

A custom view which provides a strongly-typed document model and a custom view model with specialised HTML helpers for each model.

This view expects an input model of type DocumentViewModelTdocument, Tviewmodel. The controller CodeFirstControllerTdocument provides a convenience method DocumentView{T}(T viewModel, RenderModel renderModel) to generate this model The view has an @Model property containing the original Umbraco RenderModel and also has @Document and @ViewModel properties of types Tdocument and Tviewmodel respectively, as well as @DocumentHelper and @ViewModelHelper HTML helper properties.
