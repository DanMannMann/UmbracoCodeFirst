CodeFirstController Obsolete.

Converts IPublishedContent instances to code-first strongly-typed models and returns them as the model to a view. If no suitable model exists the Umbraco RenderModel is passed through to the view, making it safe to mix code-first and manual document types.

This controller is suitable as a replacement for the default controller in the controller resolver, allowing strongly-typed views to be created without creating a custom controller.

Views which inherit [!:UmbracoViewPage>Tdocument<] or specify @model Tdocument will work with this controller
