IContentFactory Methods

The IContentFactory type exposes the following members.

Methods
  NameDescription
GetIfExists

Gets the existing instance of this default document, if it exists in the database.

This method returns any node of the correct document type with the correct alias found at the correct tree location, otherwise null

GetOrCreate

Creates and returns the default document, or returns the existing instance if there is one.

If the node already exists it should still be returned in order to ensure its' sort order and permissions are updated.

Note: It is the implementor's responsibility to ensure content doesn't already exist. Exceptions will be thrown by Umbraco after this method ends if attempting to create content which already exists, so do check.

GetPublishedIfExists

Gets the existing instance of this default content, if it exists in the published content cache.

This method returns any node of the correct document type with the correct alias found at the correct tree location, otherwise null
