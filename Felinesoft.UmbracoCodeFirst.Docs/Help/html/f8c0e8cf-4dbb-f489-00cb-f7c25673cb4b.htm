CodeFirstControllerTdocument Class
Provides convenience methods and properties for working with code-first strongly-typed models and views. This class is an ideal base for custom strongly-typed controllers, and can be inherited for any situation where you'd normally inherit RenderMvcController.

Views which inherit CodeFirstDocumentViewTdocument, Tviewmodel will work with this controller
